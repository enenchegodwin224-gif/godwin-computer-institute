HTML Editors


A simple text editor is all you need to learn HTML.

Learn HTML Using Notepad or Visual Studio Code

Web pages can be created and modified by using professional HTML editors.

However, for learning HTML we recommend a simple text editor like Notepad (PC) or Visual Studio Code (Windows and Macs).

We believe that using a simple text editor is a good way to learn HTML.

Follow the steps below to create your first web page with Notepad or Visual Studion Code.

Step 1: Open Notepad (PC)

To open Notepad, go to your start and search for Notepad and click open.

Step 1: Open Visual Stuido Code (VS Code)

Go to Google Chrome and search for Visual Studio Code, click the first link that appears on the page and then click download. It is free you don't need to pay anything, so after downloading wait for it to install, any anything it show just click "ok" or "yes". And then you are set to use it.
Go and open the VS Code and then click create a new folder, inside that folder create a new file and name it "index.html".

Step 2: Write Some HTML

Write or copy the following HTML code into Notepad or VS Code:

<!DOCTYPE html>
<html>
<head>
<title>page </title>
</head>
<body>

<h1>My First Heading</h1>
<p>My First Paragraph.</p>

</body
</html>

Step 3: Save the HTML Page

Save the file inside the new folder you created. Select File > Save as in the VS Code menu.

Name the file "index.html" and set the encoding to UTF-8 (which is the preferred encoding for HTML files).
